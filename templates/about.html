“EasyAd” is a web-based online advertisement system which is a website which will be very much user friendly and convenient for the consumers. Through this website, service providers/vendors can easily provide their service and consumers can rent a billboard or use a vehicle for advertisement. Also, consumers can be able to see the available billboards and vehicle of all places from his home. This website will be dynamic and managed by a suitable authority. The main purpose of developing this system was to enable to rent a billboard or use a vehicle for advertisement. The system has been designed in such a way that it is very cost-efficient therefore a customer or a company outside the metropolitan cities can afford it as well. The user interface design framework has been designed to make the system user-friendly therefore a non-IT technician will be able use the system to the full extent.
